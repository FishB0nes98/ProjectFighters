Abilities

Passive: Home Run Points

Shoma can purchase different balls at the base, which he receives as items. Balls are bought with "Home Run" points. Hitting an enemy or ally grants (1) Home Run point. A kill grants (5) Home Run points. Each ball costs 10 Home Run points.
